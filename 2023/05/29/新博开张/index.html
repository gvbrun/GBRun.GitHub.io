新博开张

在GitHub上建立自己的博客

1、准备工作

  1. node.js下载地址-http://nodejs.cn
  2.  Git下载地址-Git

2、登陆Github账号在主页点击Create repository

3、注意仓库的取名格式：用户名.github.io(这将是以后的访问域名)

4、在磁盘中创建一个用来存放Github本地仓库文件的目录(d:\blog)

5、选中创建的目录(文件夹)右键选择使用Gti Bash Here打开Git命令窗口(这就是开始安装的Git工具，现在开始使用)

6、提高网络稳定 npm install -g cnpm –registry=https://registry.npm.taobao.org

7、安装hexo 输入命令：cnpm install -g hexo-cli

8、始化Hexo 输入命令：hexo init

9、生成.ssh文件 输入命令：ssh-keygen -t rsa -C “your_email@youremail.com“

10、检测是否生成.ssh文件（cd ~/.ssh），未生成继续ssh-keygen -t rsa -C “your_email@youremail.com“ 然后一直按enter即可，直到C盘用户目录下出现.ssh子目录，出现id_rsa.pub

11、git config –global user.name “自定义用户名”

12、git config –global user.email “邮箱”

13、使用记事本打开id_rsa.put文件并复制生成的key

14、在已登陆的Github主页点击右侧头像 》 Settings设置，点击SSH and GPG keys 》 New SSH key，将刚刚在.ssh目录下所复制id_rsa.put文件中的信息复制进key(注意空格)，在给它取个名字(随意)

15、返回 d:\bolg 输入：cd d/blog

16、输入命令：ssh -T git@github.com 并且输入yes之后，行末尾会显示你的用户名

17、用记事本打开并修改本地仓库目录下_config.yml文件

deploy:
type: git
repository: https://github.com/用户名/用户名.github.io.git
branch: master

18、此时需要安装一个上传工具
输入命令：cnpm install hexo-deployer-git**

19、创建博客文章 输入命令：hexo new “文章名称”

20、新建文章之后需要生成一遍文件
输入命令：hexo g

21、预览一遍 **
输入命令：hexo s浏览器输入：localhost:4000(ctrl + c 关闭)

21、最后直接部署到Github就可以啦
输入命令：hexo d
